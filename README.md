# Install Instructions

## Local Dev Environment
- Start your MongoDB Service
- ```npm install```
- ```npm start```

## Docker Dev Environment
- ```npm install```
- ```docker-compose build``` * only needed if you have added additional dependencies
- ```docker-compose up```
